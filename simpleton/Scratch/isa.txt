# A very basic knowledge base
#
# Based on a very simple grammar:
#
#  Symbol    Meaning 	            Example
#  ------    ------                  ----------------
#  S 	   sentence                the man walked
#  NP 	   noun phrase             a dog
#  VP 	   verb phrase             saw a park
#  PP 	   prepositional phrase    with a telescope
#  Det 	   determiner              the
#  N 	   noun                    dog
#  V 	   verb                    walked
#  P 	   preposition             in
#
# isa(input, thing) ; given input returns a thing
#
# input, thing
Mary, woman
walked, action
Pookie, cat
in, preposition
the, determiner
park, green space
